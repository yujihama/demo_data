# マスタデータ（1.master_data）

株式会社テクノプレシジョン FY2025 JSOX評価用のマスタデータを格納。
**すべての会社名・取引先名は符号式のダミー**（サンプル顧客A社〜T社、サンプル仕入先A社〜Y社、A銀行〜G銀行）。
すべてのエビデンスはこのマスタデータを参照して整合性を保つ。

## コード体系

### 顧客コード
- `C-10001 ~ C-10007`：自動車業界（サンプル顧客A社〜G社）
- `C-10011 ~ C-10018`：半導体装置業界（サンプル顧客H社〜O社）
- `C-10021 ~ C-10025`：商社（サンプル顧客P社〜T社）

### 仕入先コード
- `V-20001 ~ V-20015`：原材料系（サンプル仕入先A社〜O社）
- `V-20021 ~ V-20028`：外注加工（サンプル仕入先P社〜W社）
- `V-20031 ~ V-20032`：消耗品その他（サンプル仕入先X社、Y社）

### 製品コード
- `P-30001 ~ P-30010`：切削加工品（自動車向け）
- `P-30011 ~ P-30020`：切削加工品（半導体装置向け）
- `P-30021 ~ P-30030`：プレス加工品

### 従業員番号 / ユーザID
- `E0001` 系：社員番号（連番）
- `CEO/CFO/COO/ACC/SLS/PUR/MFG/WHS/IT/HR/GA/IA`：部門略号付ユーザID

### トランザクションID（エビデンスで使用）
- 受注：`ORD-2025-NNNN`
- 発注：`PO-2025-NNNN`
- 検収：`REC-2025-NNNN`
- 稟議（ワークフロー）：`W-2025-NNNN`
- 仕訳：`JV-YYYYMM-NNNN`
- 変更管理申請：`REL-2025-NNN`

### 銀行（取引銀行）
A銀行、B銀行、C銀行、D銀行、E銀行、F銀行、G銀行（すべてダミー）

## 金額レンジ（連結売上380億円前提）

- 受注平均：200万～3,000万円（月間200-300件）
- 発注平均：50万～1,500万円（月間300件）
- 固定資産取得：100万～5,000万円
- 与信限度額：30,000,000～500,000,000円（顧客格付・規模による）
