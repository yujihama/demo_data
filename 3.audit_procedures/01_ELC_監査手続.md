## 1. ELCの特徴と評価アプローチ

全社統制（ELC）は、COSOの6つの構成要素（統制環境、リスク評価、統制活動、情報と伝達、モニタリング、IT対応）に基づき、**会社全体の統制環境**を評価する。
42項目ベースを採用している本社だが、本評価ではコアとなる12項目に焦点を当てる。

**評価方法**:
- 整備状況評価（規程・組織・文書の整備状況）
- 運用状況評価（会議・レビュー等の実施状況）
- チェックリスト形式（各項目「有効/不十分/未整備」で判定）

## 3. 整備状況評価手続

### 3-1. 実施要領
- 2025年7月に実施
- 各規程（R01-R27）の最新版を入手・精読
- 関連部門長へのヒアリング（CFO、総務部長、人事部長、情シス部長、内部監査室長）
- 組織図・職務権限規程の精読

### 3-2. 整備面のチェック項目

**統制環境（ELC-001~003）**
- ✓ 取締役会規則が整備され、月次開催の体制あり
- ✓ 倫理綱領が全役職員に周知され、年1回確認
- ✓ 職務権限規程が整備され、金額別承認権限が明文化

**リスク評価（ELC-004~005）**
- ✓ 年次リスクアセスメントのプロセスが整備
- ✓ 不正リスク評価シートのフォーマットあり

**統制活動（ELC-006~007）**
- ✓ 主要業務規程R11-R19が整備済
- △ **職務の分離について**: SAPロール設計に基づくSoDマトリクスは整備されているが、一部例外あり（後述）

**情報と伝達（ELC-008~009）**
- ✓ 内部通報規程および窓口（社内＋外部弁護士）が整備
- ✓ 月次決算報告のスケジュール・様式が確立

**モニタリング（ELC-010~011）**
- ✓ 内部監査規程・計画が策定
- ✓ 監査等委員会規則に基づく月次開催体制

**IT対応（ELC-012）**
- ✓ 情報セキュリティ基本方針を年1回取締役会で承認

### 3-3. 整備面の結論
すべて整備状況は有効。

## 4. 運用状況評価手続

### 4-1. 統制ごとの運用状況評価

#### ELC-001 取締役会の機能 — 12件（月次）全数検証
- 議事録の入手と確認
- 各月の開催実績、出席率、決議事項
- **結果**: 12回とも実施、すべて社外取締役3名出席 → 有効

#### ELC-002 倫理綱領の浸透 — 全役職員に対する確認
- 年1回の受領確認書の提出状況
- 提出率：100%
- 研修受講記録
- **結果**: 有効

#### ELC-003 職務権限と組織体制 — 年1回の見直し
- 2025年度見直し結果（2025/4/1付）の確認
- **結果**: 有効

#### ELC-004 全社リスク評価 — 年1回実施
- 2025/6実施のリスクアセスメント結果を確認
- 取締役会報告議事録
- **結果**: 有効

#### ELC-005 不正リスク評価 — 年1回実施
- 2025/7実施の評価シートと結果報告書を確認
- **結果**: 有効

#### ELC-006 規程・マニュアル整備 — 年1回見直し
- 改訂履歴表を確認、全規程が1年以内に見直し済
- **結果**: 有効

#### ELC-007 職務の分離 — 月次SoDレビュー 12件 【一部不備】
- 月次SoD違反レポートのレビュー記録を確認
- 11件中 職務分掌違反1件（PUR004 山田純一）が検出・未解消状態
  - 詳細：PO_CREATE と PO_APPROVE の両方が付与されており、PLC-P-002の不備の原因となっている
  - 2025/10月以降のレポートで検出されていたが、是正が遅延
- **結果**: 一部不備あり（是正計画策定中、PLC-P-002の是正と連動）

#### ELC-008 内部通報制度
- 通報受付台帳の確認
- Q2に1件受領（処理完了）
- **結果**: 有効

#### ELC-009 決算情報の伝達 — 月次12回
- 月次決算速報の経営会議報告の議事録を確認
- **結果**: 有効

#### ELC-010 内部監査の実施
- 2025年度監査計画の実施状況
- 4部門・3拠点の監査を完了
- **結果**: 有効

#### ELC-011 監査等委員会のモニタリング — 月次12回
- 議事録の入手、実施状況の確認
- **結果**: 有効

#### ELC-012 IT戦略と情報セキュリティ方針
- 2025/4取締役会での方針承認記録
- 年次IT計画書の確認
- **結果**: 有効
