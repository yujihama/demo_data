#### ◆ PLC-P-002 発注承認（金額別） — 25件サンプル検証 【真の不備】

**【25件サンプル検証手続】**

**ステップ1: 母集団リストの取得**
- SAP ME2N（発注伝票一覧）を使用
- 抽出条件：発注日 2025/4/1～2026/3/31、発注タイプ=通常発注
- 抽出日時：2026/2/12 14:20 JST
- 件数：3,874件

**ステップ2: サンプル抽出**
- 抽出間隔：3,874 ÷ 25 = 154.96 → 155件ごと
- 開始位置：89番目
- 結果をサンプル一覧.xlsxに記録

**ステップ3: 各サンプルの検証**

承認権限の判定基準：
- 購買担当（MM_USER, PO_CREATE）：50万円以下
- 購買部課長（林 真由美 E0032）：500万円以下
- 購買部長（木村 浩二 E0031）：2,000万円以下
- 管理本部長（渡辺 正博 E0002）：1億円以下
- 代表取締役：1億円超

各サンプルにつき：
- 発注金額の確認
- 承認者の権限範囲確認（employees.xlsxの承認権限金額と照合）
- 承認日≦発注日の確認
- SAPワークフロー履歴の確認

**ステップ4: 例外事項の集計**

| サンプル№ | 発注ID | 金額(円) | 承認者 | 承認者上限(円) | 判定 |
|---------|---------|---------|--------|-------------|------|
| ... 22件は合格 ... | | | | | |
| **№7** | PO-2025-0789 | **520,000** | 林 真由美（500万円上限） | OK | **要確認** |

※ サンプル7は金額が課長上限内であるため、一見合格に見える。
しかし、さらに詳細調査した結果：

| サンプル№ | 発注ID | 金額(円) | 実際の承認者 | 問題 | 判定 |
|---------|---------|---------|------------|------|------|
| **№2** | PO-2025-0234 | **680,000** | 清水 智明（PUR003、上限100万円） | 一見合格だが、SAPロール確認：PUR003はPO_CREATEのみ、PO_APPROVE権限なし | **不合格** |
| **№12** | PO-2025-1456 | **1,250,000** | 山田 純一（PUR004） | 購買部担当（上限50万円）が権限なしで承認 | **不合格** |
| **№20** | PO-2025-2890 | **7,850,000** | 林 真由美（E0032、上限500万円） | 上限500万円の課長が785万円の発注を承認 | **不合格** |

**追加調査**:
- 3件とも是正記録（上位承認の事後取得）が見つからない
- PUR004（山田 純一）に誤ってPO_APPROVE権限が付与されている（SoD違反）
- 関連する統制 ELC-007 職務の分離 にも影響

**ステップ4 集計**:
- 合格：22件
- **不合格：3件**
- 不合格率：12%

**ステップ5: 結論**
- 不合格3件以上 → **運用評価：不備**
- 想定誤謬金額：発注総額約20億円 × 12% = 約2.4億円
- 質的要素：職務分掌違反、是正記録なし
- **不備認定：重要な不備の可能性**
- 補完統制（PLC-P-004 3-wayマッチ、ITGC-AC-004 特権ID管理等）を考慮
- 最終認定は監査法人協議のうえ決定

**是正措置**:
1. PUR004（山田 純一）のPO_APPROVE権限を即時削除（2026/2/15実施済）
2. 3件について事後承認を取得（管理本部長）
3. 再発防止：SAPワークフローに金額×権限の整合性チェックを強化（ITGC-CM-001経由）
