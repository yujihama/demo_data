# 株式会社テクノプレシジョン FY2025 J-SOX評価ドキュメント一式

本ディレクトリは、架空の日本上場製造業「株式会社テクノプレシジョン」に対する
J-SOX（日本版SOX法）内部統制評価の文書セットを格納する。

- **評価対象期間**：FY2025（2025年4月1日 ～ 2026年3月31日）
- **評価主体**：株式会社テクノプレシジョン 内部監査室
- **想定監査法人**：外部監査人（独立監査人）

| [5.test_results/](5.test_results/) | **監査人の調書**：評価結果・サンプル判定・ヒアリング記録・不備管理台帳等 |

### エビデンスと監査調書の切り分け

| 区分 | 格納先 | 例 |
|------|--------|-----|
| **エビデンス**（被評価部門から入手する素材） | `4.evidence/` | SAPエクスポート、注文書PDF、画面スクリーンショット、経理部作成の突合表、議事録 等 |
| **監査人の調書**（監査人が作成した評価記録） | `5.test_results/` | 25件サンプル判定シート、例外事例メモ、ヒアリング記録、不備管理台帳 等 |

### 会社名・取引先名について

本ドキュメントセットのすべての取引先名は**完全に符号式のダミー**：
- 顧客：サンプル顧客A社 〜 サンプル顧客T社（20社）
- 仕入先：サンプル仕入先A社 〜 サンプル仕入先Y社（25社）
- 銀行：A銀行 〜 G銀行（7行）
- 従業員：姓名は架空の組み合わせ

## 意図的に仕込まれた例外・不備

本ドキュメントセットは、現実の監査で遭遇する複雑さを反映するため、以下の例外・不備を意図的に仕込んでいる。

### 【真の不備（Deficiency）】運用評価で「不合格」となる
| 統制ID | 不備の内容 | 種別 |
|--------|-----------|------|
| PLC-P-002 発注承認 | 25件中3件で承認権限超過（¥500,000上限の課長が¥520,000承認等） | 重要な不備の可能性 |
| ITGC-AC-003 退職者アカウント | 退職者5名中2名がSAP停止11日/18日遅延 | 重要な不備 |
| PLC-I-002 棚卸差異調整 | ¥850,000の棚卸差異について原因分析なし・経理報告なし | 軽微な不備 |

### 【判断保留】エビデンス不足で追加入手が必要
| 統制ID | エビデンス不足の内容 |
|--------|--------------------|
| PLC-S-005 売掛金年齢分析 | 承認印のPDFスキャンが低解像度で判読不能 |
| ITGC-AC-002 アクセス権定期棚卸 | 抽出条件・タイムスタンプ等エビデンスの完全性証跡なし |
| FCRP-003 会計上の見積レビュー | 貸倒引当金の根拠資料（信用情報等）が未添付 |

### 【軽微な例外（許容）】判断により許容・不備と認定しない
- PLC-S-001 受注承認：サンプル14番で承認日が1日遅れ（業務影響なし）
- ITGC-CM-002 変更管理：緊急変更1件が事後承認（緊急変更手順に則る）
