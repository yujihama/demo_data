## 4. 主要業務プロセス（PLC対象）

### 4-1. 販売プロセス

```
受注 → 出荷指示 → 出荷・納品 → 売上計上 → 請求 → 入金消込
```

| 段階 | 担当部門 | 使用システム | 主要証憑 |
|------|----------|-------------|---------|
| 受注登録 | 営業部 | 販売管理モジュール（ERP） | 注文書、受注伝票 |
| 与信チェック | 営業部/経理部 | ERP（自動与信限度チェック） | 与信限度マスタ |
| 出荷指示 | 営業部 | ERP | 出荷指示書 |
| 出荷・検品 | 物流倉庫 | WMS（倉庫管理システム） | 出荷報告書、納品書 |
| 売上計上 | 経理部 | ERP（出荷連動自動仕訳） | 売上伝票 |
| 請求書発行 | 経理部 | ERP（月次自動発行） | 請求書 |
| 入金消込 | 経理部 | ERP＋FBデータ取込 | 入金明細、消込リスト |

**関連する主要勘定科目：** 売上高、売掛金、受取手形

### 4-2. 購買プロセス

```
購買依頼 → 発注 → 入荷・検収 → 買掛計上 → 支払
```

| 段階 | 担当部門 | 使用システム | 主要証憑 |
|------|----------|-------------|---------|
| 購買依頼 | 製造部/各部門 | ERP（購買依頼画面） | 購買依頼書 |
| 発注承認 | 購買部（金額別承認権限） | ERP（ワークフロー） | 発注書 |
| 入荷・検収 | 購買部/品質保証部 | ERP＋検収入力 | 検収報告書 |
| 買掛計上 | 経理部 | ERP（検収連動自動仕訳） | 仕入伝票 |
| 支払 | 経理部 | ERP＋FB連携 | 支払一覧表、振込データ |

**関連する主要勘定科目：** 仕入高、買掛金、支払手形、原材料

### 4-3. 在庫管理プロセス

```
入庫 → 保管・棚卸 → 出庫 → 原価振替 → 期末評価
```

| 段階 | 担当部門 | 使用システム | 主要証憑 |
|------|----------|-------------|---------|
| 入庫登録 | 倉庫管理/製造部 | ERP＋WMS | 入庫伝票 |
| 実地棚卸（年2回） | 全社横断チーム | 棚卸リスト（ERP出力） | 棚卸報告書、差異分析 |
| 出庫・払出 | 製造部 | ERP | 出庫伝票 |
| 原価計算 | 経理部 | ERP（標準原価＋差異分析） | 原価計算表 |
| 期末評価損 | 経理部 | ERP＋Excel | 評価損計算シート |

**関連する主要勘定科目：** 棚卸資産（原材料・仕掛品・製品）、売上原価

### 4-4. 固定資産管理プロセス

```
取得申請 → 承認 → 登録 → 減価償却 → 除却・売却
```

| 段階 | 担当部門 | 使用システム |
|------|----------|-------------|
| 取得申請・承認 | 各部門/管理本部 | 稟議システム（ワークフロー） |
| 資産登録 | 経理部 | ERP固定資産モジュール |
| 減価償却計算 | 経理部 | ERP（月次自動計算） |
| 実査（年1回） | 経理部＋各部門 | 資産台帳（ERP出力） |

**関連する主要勘定科目：** 有形固定資産、減価償却費

## 7. 決算財務報告プロセスの概要

```
月次決算（経理部）
  → 四半期決算（経理部 + 各子会社経理）
    → 連結パッケージ回収・検証（経理部）
      → 連結決算（経理部）
        → 開示書類作成（経理部 + 経営企画部）
          → 取締役会承認
            → 監査等委員会レビュー
              → 外部監査人監査
```

### 重要な会計上の見積り

| 項目 | 内容 | リスク |
|------|------|--------|
| 棚卸資産評価 | 滞留在庫の評価損計算（回転期間基準） | 評価基準の恣意的適用 |
| 貸倒引当金 | 一般債権（実績率）＋個別（個別見積） | 回収可能性の過大評価 |
| 固定資産減損 | 事業用資産のグルーピング＋減損テスト | 将来CFの楽観見積 |
| 退職給付 | 数理計算上の仮定（割引率・昇給率等） | 仮定の不適切な設定 |
| 税効果 | 将来課税所得見積によるDTA回収可能性 | タックスプランニングの合理性 |

## 8. 内部統制上の主要リスク（トップリスク）

| No. | リスク領域 | 具体的リスク | 影響を受ける勘定 |
|-----|-----------|------------|----------------|
| TR01 | 売上の実在性・期間帰属 | 出荷基準の不適切適用（期末の押込み出荷） | 売上高、売掛金 |
| TR02 | 原価計算の正確性 | 標準原価の未更新による差異拡大 | 棚卸資産、売上原価 |
| TR03 | 在庫の実在性 | 帳簿棚卸と実地棚卸の乖離 | 棚卸資産 |
| TR04 | 購買の正当性 | 架空発注、不正なキックバック | 買掛金、原材料 |
| TR05 | 固定資産の網羅性 | 資産の未登録、除却漏れ | 有形固定資産 |
| TR06 | 不正なシステムアクセス | 職務分掌違反によるデータ改竄 | 全勘定 |
| TR07 | プログラム変更リスク | 未承認のプログラム変更による計算誤り | 全勘定 |
| TR08 | 連結決算の正確性 | 連結パッケージの報告誤り、内部取引消去漏れ | 連結財務諸表全体 |
